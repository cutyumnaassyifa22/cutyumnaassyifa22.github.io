Menginstall Ruby dan Jekyll

20 Feb 2025 -

Penjelasan tentang instalasi Ruby dan Jekyll.

Jekyll

1. Instalasi Ruby

Ruby adalah bahasa pemrograman yang digunakan oleh Jekyll untuk membangun situs statis. Untuk mulai menggunakan Jekyll, pastikan Ruby telah terpasang di sistem Anda.

1.1 Cek Ruby

ruby -v

1.2 Instalasi Ruby Sesuai OS

Windows: Gunakan RubyInstaller dan ikuti petunjuk instalasinya.

macOS:

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
brew install ruby
export PATH="/usr/local/opt/ruby/bin:$PATH"

Linux (Ubuntu):

sudo apt update
sudo apt install ruby-full

2. Instalasi Jekyll

gem install jekyll bundler

2.1 Buat Proyek Jekyll

jekyll new my-site

2.2 Jalankan Situs Lokal

cd my-site
bundle exec jekyll serve

2.3 Struktur Direktori

  • _posts/: Menyimpan artikel blog
  • _config.yml: Konfigurasi situs
  • index.html: Halaman utama

3. Menggunakan Bundler

Bundler membantu mengelola dependensi Ruby.

gem install bundler

3.1 Buat File Gemfile

source 'https://rubygems.org'
gem 'jekyll'
gem 'jekyll-feed'

3.2 Instalasi dengan Bundler

bundle install

4. Kesimpulan

  • Ruby: Dibutuhkan untuk menjalankan Jekyll.
