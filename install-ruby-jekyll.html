Menginstall Ruby dan Jekyll

20 Feb 2025

Penjelasan tentang instalasi Ruby dan Jekyll.

1. Instalasi Ruby

Ruby adalah bahasa pemrograman yang dinamis dan berorientasi objek. Untuk mengembangkan situs dengan Jekyll, Anda memerlukan Ruby sebagai bagian dari persyaratan instalasi.

1.1. Langkah-langkah Instalasi Ruby

A. Cek apakah Ruby sudah terinstal

ruby -v

B. Instalasi Ruby di macOS

Menggunakan Homebrew:

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
brew install ruby

Menambahkan Ruby ke PATH:

export PATH="/usr/local/opt/ruby/bin:$PATH"
source ~/.zshrc   # Untuk pengguna zsh
source ~/.bash_profile  # Untuk pengguna bash

C. Instalasi Ruby di Windows

Gunakan RubyInstaller, centang opsi "Add Ruby executables to your PATH", lalu verifikasi:

ruby -v

D. Instalasi Ruby di Linux (Ubuntu/Debian)

sudo apt update
sudo apt install ruby-full
ruby -v

2. Instalasi Jekyll

2.1. Langkah-langkah Instalasi Jekyll

gem install jekyll bundler
jekyll -v

Membuat situs baru:

jekyll new my-site

Menjalankan server lokal:

cd my-site
bundle exec jekyll serve

Akses di: http://localhost:4000

Menyunting situs:

  • _posts/: Artikel blog
  • _config.yml: Konfigurasi
  • index.md: Halaman utama

3. Menggunakan Bundler untuk Manajemen Dependensi

3.1. Instalasi Bundler

gem install bundler

3.2. Menggunakan Bundler

source 'https://rubygems.org'

gem 'jekyll'
gem 'jekyll-feed'
bundle install
bundle exec jekyll serve

4. Kesimpulan

Instalasi Ruby dan Jekyll bisa dilakukan sesuai sistem operasi. Gunakan Bundler untuk mengatur dependensi proyek. Dengan mengikuti langkah-langkah di atas, Anda bisa mulai membangun situs statis dengan Jekyll.
