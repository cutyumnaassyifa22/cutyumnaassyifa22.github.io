HTML Link and Lists
20 Mar 2025

HTML Link digunakan untuk membuat tautan ke halaman lain dengan tag <a>:

<a href="https://example.com">Klik di sini</a>

HTML Lists digunakan untuk menampilkan daftar item:

  • <ul>: daftar tidak berurutan (bullet)
  • <ol>: daftar berurutan (angka)
  • <li>: item di dalam list
Layout dan Navigation
13 Mar 2025

Layout menyusun bagian-bagian HTML seperti:

  • <header>: bagian atas
  • <nav>: menu navigasi
  • <main>: isi utama
  • <footer>: bagian bawah

Navigation menggunakan link <a> untuk berpindah antar halaman:

<nav>
  <a href="index.html">Home</a>
  <a href="blog.html">Blog</a>
</nav>
Markdown
06 Mar 2025

Markdown adalah bahasa ringan untuk membuat format dokumen.

Contoh sintaks Markdown:

# Judul
## Subjudul
- List item
- List lainnya
[Link](https://example.com)

Markdown sering digunakan di GitHub README.md dan dokumentasi lainnya.

Menginstall Ruby dan Jekyll
20 Feb 2025

Langkah instalasi Ruby & Jekyll:

  1. Download Ruby dari rubyinstaller.org
  2. Install bundler dan jekyll: gem install bundler jekyll
  3. Buat proyek baru: jekyll new mysite
  4. Masuk folder proyek: cd mysite
  5. Jalankan: bundle exec jekyll serve

Lalu akses di browser: http://localhost:4000
