HTML Link and Lists

Link HTML digunakan untuk membuat tautan ke halaman lain menggunakan tag <a>:

<a href="https://example.com">Klik di sini</a>

List HTML adalah daftar yang bisa berupa:

  • <ul> untuk daftar tidak berurutan (bullet)
  • <ol> untuk daftar berurutan (angka)
<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>

Layout dan Navigation

Layout digunakan untuk menyusun struktur halaman:

  • <header> - bagian atas
  • <nav> - navigasi
  • <main> - konten utama
  • <footer> - kaki halaman

Navigation biasanya ditulis seperti ini:

<nav>
  <a href="index.html">Home</a>
  <a href="blog.html">Blog</a>
</nav>

Markdown

Markdown adalah bahasa ringan untuk menulis dokumen web. Contoh:

# Judul Besar
## Subjudul
- List item
- List item
[Link](https://google.com)

Markdown sering dipakai di GitHub (README.md), dokumentasi, dll.

Menginstal Ruby dan Jekyll

Langkah Instalasi Ruby dan Jekyll:

  1. Download Ruby: rubyinstaller.org
  2. Buka terminal: gem install jekyll bundler
  3. Mulai proyek: jekyll new mysite
  4. Masuk folder: cd mysite
  5. Jalankan lokal: bundle exec jekyll serve

Buka di browser: http://localhost:4000
