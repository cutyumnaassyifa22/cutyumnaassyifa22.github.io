    HTML Link and Lists

    HTML Links digunakan untuk membuat tautan antara halaman-halaman web. Tautan ini diwakili oleh tag <a>. Contoh dasar dari HTML Link adalah:

    <a href="https://www.example.com">Kunjungi Example</a>

    Di sini, href adalah atribut yang menentukan alamat URL tujuan. Tautan ini dapat berupa link internal (ke halaman lain dalam situs yang sama) atau eksternal (ke situs lain).

    HTML Lists digunakan untuk mengelompokkan item dalam urutan tertentu. Ada dua jenis utama: unordered list dan ordered list.

    • Unordered list menggunakan tag <ul> dan item ditandai dengan tag <li>.
    • Ordered list menggunakan tag <ol>, dan itemnya juga ditandai dengan <li>.

    Penjelasan tentang Layout dan Navigation

    Layout dan Navigation

    Layout dan navigasi penting dalam membangun sebuah website yang terstruktur dengan baik. Layout mengacu pada bagaimana elemen-elemen dalam halaman web ditempatkan dan diatur. Navigasi mengacu pada cara pengguna dapat berpindah antar halaman atau bagian di dalam website.

    Contoh dasar penggunaan layout dengan HTML dapat dilakukan dengan tag <header>, <nav>, <section>, dan <footer>.

    <header>Ini adalah header</header>
    <nav>Ini adalah navigasi</nav>
    <section>Konten utama halaman</section>
    <footer>Footer halaman</footer>

    Penjelasan tentang Markdown

    Markdown

    Markdown adalah bahasa markup ringan yang digunakan untuk menulis teks yang dapat dengan mudah dikonversi menjadi HTML. Markdown lebih mudah dibaca dan ditulis daripada HTML biasa, dan banyak digunakan dalam dokumentasi, blog, dan forum.

    Contoh penulisan Markdown:

    # Judul 1
## Judul 2
- Item 1
- Item 2
[Link ke Google](https://www.google.com)

    Markdown memungkinkan kamu menulis teks dengan format yang lebih sederhana dan lebih bersih daripada HTML. Setiap heading diawali dengan tanda pagar (#), dan daftar dimulai dengan tanda minus (-).

    Penjelasan tentang instalasi Ruby dan Jekyll

    Menginstal Ruby dan Jekyll

    Ruby adalah bahasa pemrograman yang digunakan oleh Jekyll, yang merupakan framework statis untuk membuat situs web. Untuk memulai, kamu perlu menginstal Ruby terlebih dahulu.

    Langkah-langkah untuk menginstal Ruby dan Jekyll di sistem kamu:

    1. Unduh dan instal Ruby dari situs resmi Ruby: https://www.ruby-lang.org/en/documentation/installation/
    2. Setelah Ruby terinstal, buka terminal atau command prompt dan jalankan perintah berikut untuk menginstal Jekyll:
      3. gem install jekyll bundler
    3. Setelah instalasi selesai, kamu dapat membuat proyek Jekyll baru dengan perintah:
      4. jekyll new my-awesome-site
    4. Masuk ke direktori proyek Jekyll:
      5. cd my-awesome-site
    5. Jalankan situs Jekyll lokal dengan perintah:
      6. bundle exec jekyll serve

    Setelah itu, situs Jekyll kamu akan tersedia di http://localhost:4000.
